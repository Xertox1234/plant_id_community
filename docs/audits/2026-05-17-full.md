## Open Question ‚Äî Forum Configuration (blocks triage of ~18 findings)

`settings.py:199-204`: `apps.forum` (the headless DRF forum ‚Äî viewsets, models,
16 test modules, the app `CLAUDE.md` calls the *active* forum) is appended to
`LOCAL_APPS` **only in the `else` branch** of `if ENABLE_FORUM:`. `backend/.env`
sets `ENABLE_FORUM=True`, so the **Machina** forum is active and `apps.forum`
is **not installed**. Findings against `apps/forum/*` (H14, H15, H35, M9, M11,
M13, M46, L6, L31, plus several test-quality items) are only "live" if
`apps/forum/` is the intended forum. **The user must confirm which forum is
authoritative before these are triaged.**
